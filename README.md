# DrumSet

DrumSet is a basic vanilla JavaScript project.

## Installation

Access the site from the URL below.

```bash
https://reetpriye.github.io/drumset
```

## Usage

```bash
Press w,a,s,d,j,k,l for the corresponding sound. Also you can use mouse click.
```
